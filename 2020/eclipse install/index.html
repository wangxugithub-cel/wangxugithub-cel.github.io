上篇说到安装JDK 安装好JDK就可以编程了 现在我介绍下原始的编程
其实在电脑的记事本上就可以编程的：
新建一个记事本，把记事本的后缀改为Java如：


在HelloWorld.java文件上编辑

public class HelloWorld{//注意class后面的类名必须和外面的文件名是一致的
public static void main (String [] args){
System.out.println(“hello world”);
}
}

这样一个简单的HelloWorld就写好了，那么现在开始编译
window键+R键调出命令提示符 输入 cmd 打开命令提示符
看你刚刚的Java文件放在哪里了 我的是在D盘里 所以输入
D： 进到D盘如下图：

在D盘输入javac HelloWorld.java:

在刚刚存java文件的里面看看是不是有一个HelloWorld.class
的文件如：

最后在cmd中输入java HelloWorld （注意不需要写.java后缀）
最后在cmd上输出的就是上面写的hello world的内容如：


OK简单的在记事本上编辑了一个简单的helloworld那么现在开始
